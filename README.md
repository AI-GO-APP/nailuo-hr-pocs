# 耐落（NYLOK）HR POC 集

針對耐落集團（B2B 製造業 · 螺絲防鬆膠塗佈加工）所製作的兩套 AI 賦能 HR POC。

| POC | 解決問題 | 主要對象 |
|---|---|---|
| 🏖️ [請假系統](./leave-system/) | 用對話請假 + AI 自動驗證規則 + 主管自動審核 | 全體員工 + 部門主管 |
| ⚠️ [客戶流失風險分析](./churn-analysis/) | AI 讀業務日誌、找出可能流失的客戶 | 業務人員 + 業務主管 |

## 🚀 快速開始

### 方法一：本機伺服器（推薦）

```bash
# 在這個 repo 根目錄
python3 -m http.server 8800

# 瀏覽器打開
open http://localhost:8800/
```

### 方法二：直接開檔

雙擊任一個 `.html` 即可，所有 CSS / JS / 圖示都 inline，**完全不需網路**。

## 🏖️ 請假系統

依據 **HR-TW-P-001-33 請假控制程序**（台灣廠區）建構：

### 員工端（`employee-chat.html`）
- 💬 **AI 對話請假** — 用聊天請假，AI 即時檢查額度、提醒證明、找核決主管
- 📋 **我的請假紀錄** — 含狀態徽章、篩選、KPI
- 🗓️ **出勤打卡** — 即時時鐘、工時自動計算、打下班卡
- 📊 **我的假期額度** — 12 種假別額度進度條
- 📖 **請假規範說明** — 12 張假別 policy 卡，含薪資/證明/期限規則
- 👥 **我的代理人** — 主代理人 / 副代理人 / 我代理過的同事

### 主管端（`manager-dashboard.html`）
- 📊 **即時總覽** — 4 KPI + 5 件待簽核（卡片可點開詳細）+ 32 人出勤地圖 + AI 洞察
- ⏳ **待簽核** — 完整列表 + 點任一行/查看可開詳細 modal
- 📅 **今日出勤** — 誰在崗、誰請假、代理人是誰
- 📈 **請假分析** — 假別甜甜圈、一週分布、6 個月趨勢
- ⚠️ **異常出勤** — 7 種 AI 監測模式 + 3 位高風險員工
- 👥 **員工請假履歷** — 全部 32 人完整紀錄
- 📖 **請假規則** — 18 種假別規則 + 4 種不扣薪資格
- ⚙️ **系統設定** — 通知 / AI 偵測模型 / 數據連線 / 區域權限

### 請假單詳細 Modal（亮點）
點任何待簽核項目可開啟詳細 modal，包含：
- 申請人完整資訊
- 證明文件預覽（已附 / 缺件 / 不需）
- 🤖 **AI 自動驗證**（額度檢查 / 證明完整 / 代理人狀態 / 異常偵測）
- 核決流程視覺化（多階段呈核）
- 申請人本年度請假概況
- 一鍵批准 / 退回（含確認對話）

## ⚠️ 客戶流失風險分析

依據真實 CRM 資料（4050 筆業務日誌 / 747 個客戶 / 81 位業務）製作：

### 分析方法（hybrid）
1. **規則粗篩**（`score.py`）— 關鍵字 + 上下文 pattern 命中
2. **AI 精判**（`judgments.py`）— Claude 親自讀 150 筆候選日誌，給出 risk 分數 + 類別 + 理由
3. **聚合輸出**（`finalize.py`）— 客戶層級綜合分 + 異常處理次數加權

### 主管端（`manager-dashboard.html`）
- 📊 **風險總覽** — 4 KPI + 15 個高風險客戶排名 + 風險類別分布
- 👥 **業務分析** — 業務人員風險地圖（散點圖）+ Top 業務排名 + 客戶等級分布 + 行動建議
- 🎯 **風險類別分析**（5 個類別 tab）
  - 競爭搶單 → 競爭對手點名（越南廠商、特耐、國林、和諧、春秋…）
  - 品質客訴 → 品質問題類型分布（鼓包、頭部相連、生鏽、漏點膠…）
  - 營運下滑 → 終端品牌影響地圖（北汽 -70%、奇瑞 -30%、小米…）
  - 帳款問題 → 帳款警訊分布
  - 關係惡化 → 關係警訊類型
- 🔗 **資料來源** — 連線狀態 + 原始日誌預覽 + 更新分析按鈕

### 業務端（`sales-input.html`）
- 📝 **填寫日誌** — 含 4 種情境快速範例 chip
- 🧠 **AI 即時判讀** — 輸入工作描述 800ms 後自動跳出 Risk 分數
- 📋 **我的歷史日誌** — 28 筆紀錄 + 主管動態
- 👥 **我的客戶** — 12 個客戶 + AI 風險提醒

## 🎨 技術細節

- **零依賴** — 所有 CSS / JS / 圖示都 inline 在 HTML 中
- **零外部 CDN** — 不依賴 Tailwind、Chart.js 等線上資源（離線可跑）
- **單一 HTML 檔** — 每個 POC 都是獨立檔案，可單獨寄送
- **設計風格** — 參考 Lia demo（白底 sidebar、藍紫漸層、卡片式佈局）
